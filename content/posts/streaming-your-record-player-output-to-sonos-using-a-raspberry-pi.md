<p>Got an old Raspberry Pi laying around? Into Vinyl? Thought about how you might be able to stream your music without shelling out &gt;£400 for the Sonos Connect Stereo Adaptor?</p><p>This might be the tutorial for you!</p><p>I've been into Vinyl for as long as I can remember. And having recently purchased <a href="https://amzn.to/3rb4nfa">Sonos One</a>'s for around my home, the question has come up. How can I stream my music to Sonos as I would with Spotify or Tidal?</p><p>It's worth noting that this tutorial can be used for projects that don't involve a Record Player or Sonos. You can use the Raspberry Pi to stream any audio to any endpoint. The tool that we will be using will produce a locally accessible URL which is your stream. </p><h2 id="parts-list">Parts List</h2><p>For this to work, you're going to need a few bits.</p><ul><li>A <a href="https://thepihut.com/collections/raspberry-pi/products/raspberry-pi-4-model-b?variant=20064052740158">Raspberry Pi</a> (Most generations will work, only requirement is a USB port and an Ethernet Port, unless you have a WiFi Adaptor or WiFi enabled Pi)</li><li><a href="https://amzn.to/3xaqwxN">SD Card</a></li><li>Power Lead</li><li><a href="https://amzn.to/3uvNxK9">Ethernet Cable</a> (Only if your Pi doesn't support WiFi)</li><li><a href="https://amzn.to/35ROra2">RCA to USB Interface</a> (This is used to grab the audio output from your record player as RCA. You might not need this if your record player provides USB output)</li></ul><h2 id="getting-started">Getting Started</h2><p>We're going to be setting this up in a headless manner. Headless just means that we won't need to connect a keyboard, monitor or mouse. Setting this up in a headless way does mean that we need to use a terminal to configure our setup. I'll make the steps as simple as possible so that you can copy and paste the commands. I'll also give a run down of what the commands are for.</p><h3 id="installing-ubuntu-server-onto-your-raspberry-pi">Installing Ubuntu Server onto your Raspberry Pi</h3><p>For this tutorial, I am going to be using Ubuntu Server on my Raspberry Pi. Ubuntu Server is a lightweight image which provides a headless terminal interface for you to install the required dependencies on to. You could use something like Raspberry Pi Noobs which will give you a graphical interface, but you'll then need to plug in a monitor, keyboard and mouse.</p><h3 id="step-1-downloading-the-raspberry-pi-imager">Step 1: Downloading the Raspberry Pi Imager</h3><p>In order to simply install an operating system onto our Raspberry Pi, we can use a tool called Raspberry Pi Imager. This is a tool provided by Raspberry Pi themselves in order to make installation simpler for the end user. Previously we had to download images ourselves and flash them on to our SD cards.</p><p>Head over to <a href="https://www.raspberrypi.com/software/">https://www.raspberrypi.com/software/</a> and download the Imager</p><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-1.png" class="kg-image" alt loading="lazy" width="1268" height="623" srcset="/images/size/w600/2022/04/image-1.png 600w, /images/size/w1000/2022/04/image-1.png 1000w, /images/2022/04/image-1.png 1268w" sizes="(min-width: 720px) 720px"></figure><p>Once you have the Imager downloaded. Install it onto your computer and open it. You'll be greeted with a welcome screen that looks like this:</p><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-2.png" class="kg-image" alt loading="lazy" width="684" height="455" srcset="/images/size/w600/2022/04/image-2.png 600w, /images/2022/04/image-2.png 684w"></figure><h3 id="step-2-writing-ubuntu-server-to-your-sd-card">Step 2: Writing Ubuntu Server to your SD card.</h3><p>The first thing we want to do is to choose our operating system.</p><p>Click on "CHOOSE OS".</p><p>Then, scroll down and select "Other general-purpose OS"</p><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-3.png" class="kg-image" alt loading="lazy" width="682" height="460" srcset="/images/size/w600/2022/04/image-3.png 600w, /images/2022/04/image-3.png 682w"></figure><p>Then Ubuntu</p><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-4.png" class="kg-image" alt loading="lazy" width="683" height="454" srcset="/images/size/w600/2022/04/image-4.png 600w, /images/2022/04/image-4.png 683w"></figure><p>Now we want to select an Ubuntu Server image.</p><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-5.png" class="kg-image" alt loading="lazy" width="683" height="454" srcset="/images/size/w600/2022/04/image-5.png 600w, /images/2022/04/image-5.png 683w"></figure><p>We now have the OS selected to install. It'd be a good idea to insert your SD card into your computer so that we can write the OS to the SD card.</p><p>Once inserted, we can configure the OS install. There's a little know trick with the installer which allows you to preconfigure your install prior to writing it to the SD card. I wrote more about it in <a href="https://theselfhostingblog.com/posts/issue-12-save-time-by-using-the-raspberry-pi-imagers-hidden-feature/">our self-hosting newsletter</a>.</p><p>The short of it is, with the imager window selected, hit <code>ctrl + shift + x</code> on your keyboard.</p><p>This will open up the Advanced Options window for your install.</p><p>Here you will want to enable the following:</p><ul><li><strong>Hostname </strong>(Optional, give your Pi a hostname so that you can recognise it in your network devices)</li><li><strong>SSH </strong>(set your username and password)</li><li><strong>WLAN </strong>(enter your WiFi details here, unless you're using Ethernet)</li><li><strong>Locale </strong>(set your locale and keyboard layout)</li></ul><figure class="kg-card kg-image-card"><img src="/images/2022/04/image-6.png" class="kg-image" alt loading="lazy" width="1027" height="934" srcset="/images/size/w600/2022/04/image-6.png 600w, /images/size/w1000/2022/04/image-6.png 1000w, /images/2022/04/image-6.png 1027w" sizes="(min-width: 720px) 720px"></figure><p>Finally, hit Save.</p><p>Once saved, select your SD card from the dropdown list under "CHOOSE STORAGE".</p><p>Once selected, click on "WRITE".</p><p><strong>THIS WILL WIPE ALL DATA ON THE SD CARD</strong>. Be sure to know what you're deleting, if anything.</p><h3 id="step-3-booting-up-your-raspberry-pi">Step 3: Booting up your Raspberry Pi</h3><p>Now that we have Ubuntu Server installed on an SD card, you can go ahead and insert the SD card into your Pi. Also insert your power cable and your ethernet cable (if required). You can then boot up the Pi.</p><p>Give the Pi a moment to boot up, after a minute or so you'll need to find the IP address for the Raspberry Pi. If you're familiar with networking, you can go ahead and skip this bit. If not, here's how you can find the Pi on your network.</p><h4 id="finding-the-raspberry-pi-on-your-network">Finding the Raspberry Pi on your network</h4><p>For this, you can either log into your router to get the IP address of your Pi. Or you can install an application onto your phone called <a href="https://www.fing.com/">Fing</a>.</p><p>Fing is a network scanner which will discover all devices connected to the same network as your phone. Ensure that your phone is connected to the same network as your Raspberry Pi for it to be discovered.</p><p>Once Fing is installed on your phone, run a scan and you should see your Raspberry Pi pop up. Take note of the IP address as we'll use this to connect to our Pi.</p><h3 id="step-4-sshing-into-your-raspberry-pi">Step 4: SSH'ing into your Raspberry Pi</h3><p></p><p></p><p></p>
