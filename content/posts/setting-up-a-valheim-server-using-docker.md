The problem is, someone needs to host their world.</p><p>What happens when the friend needs a break from slaying trolls or punching trees? The server needs to be shut down, as it's running from their computer.</p><p>I'm going to talk through how to host a Valheim server on a docker image. This will enable you to set up a permanent server for all your friends to enjoy, 24/7.</p><h2 id="setting-up-your-valheim-server">Setting up your Valheim server</h2><h3 id="step-1-setting-up-your-server">Step 1: Setting up your server</h3><p><a href="#step-2-installing-docker"><em>Click to jump to <strong>Step 2: Installing Docker</strong></em></a></p><p>You'll need to either have an existing server instance or create one. I use a Proxmox instance running on a server in my loft. You could also use something like <a href="https://www.digitalocean.com/">Digital Ocean</a> to run your Valheim server. Using the following link will give you $100 worth of credits for 60 days to play around with, just sign up using <a href="https://m.do.co/c/d2a3afe52625">this link</a>. That's almost 3 months of Valheim multiplayer, for free!</p><p>Once you're all set up you'll need to access your machine and run the following commands to get all your prerequisites installed!</p><h3 id="step-2-installing-docker">Step 2: Installing Docker</h3><p><em><a href="#step-3-setting-up-your-valheim-docker-server">Click to jump to <strong>Step 3: Setting up your Valheim Docker server</strong></a></em></p><p>Once you have your Ubuntu instance up and running using any of the above methods, It's time to install Docker.</p><p>Firstly, we need to run an update to make sure that all our package repositories are up to date. You can do that by executing the following in your terminal window.</p><pre><code class="language-bash">sudo apt-get update
sudo apt-get upgrade</code></pre><p>This may take up to 10 minutes, allow it to sit, go make a cup of tea or play some more Valheim!</p><p>Once your upgrade has finished, we need to install Docker. </p><p>Docker allows us to run what's called containers on our Ubuntu instance. This means, in theory, we could run multiple Valheim instances on a single machine.</p><p>Install Docker by executing the following command</p><pre><code class="language-bash">sudo apt-get install docker-ce docker-ce-cli containerd.io</code></pre><p>This will install Docker, and its required dependencies to get up and running.</p><p>Finally, the last step for the Docker setup is to assign your user to the Docker group on your machine. This allows you to run Docker commands without needing to enter your sudo password. You can add your user to the Docker group by running the following command.</p><pre><code class="language-bash">sudo usermod -aG docker $USER</code></pre><p>You will be required to log out of your Ubuntu instance and log back in again. From the terminal, this can be done by executing the following command.</p><pre><code class="language-bash">logout</code></pre><p>Once you have logged back in, you can test that your user has been assigned to the Docker group by executing the following command.</p><pre><code class="language-bash">docker -v</code></pre><p>This will give you an output of the Docker version number that you have installed. Here's my Docker version output for reference.</p><pre><code class="language-bash">~ docker -v
Docker version 19.03.8, build afacb8b7f0</code></pre><h3 id="step-3-setting-up-your-valheim-docker-server">Step 3: Setting up your Valheim Docker server</h3><p>Now it's time to set up your Valheim Docker server. </p><blockquote>You're only 15 minutes away from fighting Greydwarfs and Trolls.</blockquote><p>Make sure that you're still logged into your Ubuntu instance. Change your working directory to your users' home by running the following command.</p><pre><code class="language-bash">cd</code></pre><p>We now need to create a directory that will be the home for our Valheim server. Let's create a new directory using the <code>mkdir</code> command.</p><pre><code>mkdir valheim-server-config</code></pre><p>This will create your Valheim server directory.</p><p>You now have 2 options. Do you want to copy an existing world (say, a singleplayer world that you have been playing on, that you'd like to share with your friends)?</p><p>Or, would you like to set up a brand new world and start again?</p><p><a href="#copying-an-existing-world">I'll copy my existing world</a></p><p><a href="#creating-a-new-world">I'll create a new world for my server</a></p><h4 id="copying-an-existing-world">Copying an existing world</h4><p>To use an existing world, you will need to find your world files. On Windows, these can be found in the following location </p><pre><code>C:\Users\Ed\AppData\LocalLow\IronGate\Valheim\worlds</code></pre><p>Before copying, we need to make sure that we have created the worlds folder on our server. To do this, we need to cd into the config directory that we created earlier, and then make a new directory name <code>worlds</code>. You can achieve this by running the following commands.</p><pre><code class="language-bash">cd
cd valheim-server-config
mkdir worlds
cd</code></pre><p>So this will create the necessary <code>worlds</code> folder required to store your singleplayer world.</p><p>Now, back on your Windows machine, we need to copy all the files inside the <code>C:\Users\Ed\AppData\LocalLow\IronGate\Valheim\worlds</code> directory, over to the <code>/home/$USER/valheim-server-config/worlds</code> directory on your Ubuntu instance.</p><p>You can do this using something like <a href="https://winscp.net/eng/index.php">WinSCP</a>.</p><figure class="kg-card kg-bookmark-card"><a class="kg-bookmark-container" href="https://winscp.net/eng/index.php"><div class="kg-bookmark-content"><div class="kg-bookmark-title">WinSCP</div><div class="kg-bookmark-description">WinSCP is a popular free SFTP and FTP client for Windows, a powerful file manager that will improve your productivity. It supports also Amazon S3, FTPS, SCP and WebDAV protocols. Power users can automate WinSCP using .NET assembly.</div><div class="kg-bookmark-metadata"><img class="kg-bookmark-icon" src="https://winscp.net/favicon.ico"><span class="kg-bookmark-author">WinSCP - Free SFTP and FTP client</span></div></div><div class="kg-bookmark-thumbnail"><img src="https://winscp.net/eng/images/og_logo_240x240.jpg"></div></a></figure><p>Simply download WinSCP and log into your server using the WinSCP interface. This will allow you to upload and download files bi-directionally.</p><p>Make sure to only copy your singleplayer files, as we don't want to lose any data in the process.</p><p>Now, we want to start our new Valheim Docker server.</p><p>To do this, make sure you're back in your terminal window on your Ubuntu server. You're going to want to paste in the following Docker run command. This will start up your Docker container with some environment variables.</p><pre><code class="language-bash">docker run -d --name valheim-server -p 2456-2458:2456-2458/udp -v $HOME/valheim-server-config:/config -e SERVER_NAME="CHANGE ME" -e WORLD_NAME="CHANGE ME" -e SERVER_PASS="CHANGE ME" lloesche/valheim-server</code></pre><p>As you can see, you'll need to change some settings. </p><ul><li>Make sure to change your Server Name. This is how your friends will find your server.</li><li>You must also change the World Name to match the world name of your singleplayer world. (This is also the name of the files you copied earlier). </li><li>And finally, you'll want to add a password to your server, which <strong>must be longer than 5 characters</strong>.</li></ul><p>After you have that configured, press enter to execute the command.</p><h4 id="creating-a-new-world">Creating a new world</h4><p>Make sure you're back in your terminal window on your Ubuntu server. You're going to want to paste in the following Docker run command. This will start up your Docker container with some environment variables.</p><pre><code>docker run -d --name valheim-server -p 2456-2458:2456-2458/udp -v $HOME/valheim-server-config:/config -e SERVER_NAME="CHANGE ME" -e WORLD_NAME="CHANGE ME" -e SERVER_PASS="CHANGE ME" lloesche/valheim-server</code></pre><p>As you can see, you'll need to change some settings. </p><ul><li>Make sure to change your Server Name. This is how your friends will find your server.</li><li>You must also add a World Name, this can be anything you like.</li><li>And finally, you'll want to add a password to your server, which <strong>must be longer than 5 characters</strong>.</li></ul><p>After you have that configured, press enter to execute the command.</p><h3 id="step-4-starting-your-server">Step 4: Starting your server</h3><p>After executing the Docker run command. Navigate to the Join Game tab. You will then have to select the <code>Community</code> option.</p><p>You can now search for your server.
